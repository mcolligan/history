# Factual History

### A quick and easy history "fact" finder.

### Getting Started:

###### Install Dependancies -
```
$ npm install
```

###### Start server
```
$ npm start
```
-- Server will start on port 3000.

-- Navigate to http://localhost:3000 and begin searching.
